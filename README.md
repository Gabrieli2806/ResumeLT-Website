# ResumeLT Website

Sitio web estático bilingüe (español e inglés) para presentar el perfil profesional de **Liliana Toro** mediante una portada de selección y dos portfolios/currículums especializados.

## Función real del proyecto

Este repositorio no contiene una utilidad de desarrollo ni una herramienta para previsualizar ubicaciones de pantalla.  
Su función actual es servir como **sitio portfolio/CV** con dos perfiles:

- **Construcción**: experiencia en gestión de proyectos, supervisión de obra y coordinación de contratistas.
- **Repostería**: experiencia en hotelería, producción pastelera y liderazgo de equipos de cocina.

## Estructura del sitio

- `/home/runner/work/ResumeLT-Website/ResumeLT-Website/index.html`  
  Página principal para seleccionar el perfil.
- `/home/runner/work/ResumeLT-Website/ResumeLT-Website/arquitectura.html`  
  CV/portfolio orientado a construcción.
- `/home/runner/work/ResumeLT-Website/ResumeLT-Website/reposteria.html`  
  CV/portfolio orientado a repostería.
- `/home/runner/work/ResumeLT-Website/ResumeLT-Website/styles.css`  
  Estilos globales, layout responsive y animaciones.
- `/home/runner/work/ResumeLT-Website/ResumeLT-Website/script.js`  
  Cambio de idioma, animaciones por scroll, efectos visuales y manejo básico de imágenes.

## Características principales

- Selector de idioma **ES / EN**
- Página inicial con tarjetas para elegir perfil
- Dos páginas de CV especializadas
- Diseño responsive
- Animaciones de entrada y scroll
- Persistencia del idioma seleccionado usando `localStorage`

## Tecnologías usadas

- HTML
- CSS
- JavaScript vanilla

## Ejecución

Como es un sitio estático, no requiere build ni dependencias.  
Solo abre `index.html` en un navegador.
